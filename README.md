# Blog Archive

Personal backup and archive of my blog posts from [enerrio bearblog](https://enerrio.bearblog.dev).

## Structure

Each blog post has its own directory containing:
- `content.md` - The full blog post content in markdown
- `metadata.md` - Post metadata (title, publish date, summary, URL)
- `resources/` - Images and other media files used in the post
